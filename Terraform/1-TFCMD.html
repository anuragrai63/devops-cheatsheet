Initialization

Start Here

terraform init

Initialize directory, download plugins.

Copied!

Upgrade providers to newest allowed versions.

Copied!
terraform init -reconfigure

Reconfigure backend (ignore saved config).

Copied!

Load backend configuration from file.

Planning

Dry Run

terraform plan

Show changes required by current config.

Copied!
terraform plan -out=tfplan

Save the plan to a file for safety.

Copied!
terraform plan -target=resource

Plan only for a specific resource.

Copied!
terraform plan -refresh-only

Check for drift without changing resources.

Copied!

Execution

Make it Real

terraform apply

Create or update infrastructure.

Copied!
terraform apply -auto-approve

Apply without asking for "yes".

Copied!
terraform apply tfplan

Apply a saved plan file exactly.

Copied!
terraform apply -replace="res"

Force destroy and recreate a resource.

Copied!

Destruction

Clean Up

terraform destroy

Destroy ALL infrastructure.

Copied!
terraform destroy -target=res

Destroy only specific resources.

State Management

Backend Control

terraform state list

List all resources tracked in state.

Copied!
terraform state show <name>

Show details of a resource in state.

Copied!
terraform state mv old new

Rename resource without destroying it.

Copied!
terraform state rm <name>

Stop tracking resource (no destroy).

Copied!
terraform state pull > file

Download remote state locally.

Copied!
terraform state push file

Upload local state file to remote.

Copied!
