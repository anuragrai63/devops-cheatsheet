1. Cluster Basics

Cluster and kubeconfig management
#1
kubectl config view Copied!
Show merged kubeconfig.
kubectl config current-context Copied!
Show the current context.
kubectl config use-context my-cluster Copied!
Switch kubeconfig context to another cluster.

2. kubectl Basics

Common get/describe/apply/delete commands
#2
kubectl get pods -A Copied!
List all pods in all namespaces.
kubectl describe pod nginx-123 -n web Copied!
Detailed pod info and events.
kubectl apply -f deployment.yaml Copied!
Create/update resources from file.
kubectl delete -f deployment.yaml Copied!
Remove resources defined in file.

3. Deployments & Pods

Rolling updates, scaling and rollbacks
#3
kubectl set image deployment/web web=nginx:1.19 Copied!
Trigger a rolling update by changing image.
kubectl rollout status deployment/web Copied!
Watch rollout progress.
kubectl rollout undo deployment/web Copied!
Rollback to previous ReplicaSet.
kubectl scale deployment/web --replicas=5 Copied!
Change replica count for scaling.

4. Services & Networking

Service types, port-forwarding and ingress
#4
kubectl get svc -n web Copied!
List services in namespace.
kubectl port-forward svc/web 8080:80 -n web Copied!
Forward local port to a service for debugging.
kubectl get ingress -A Copied!
List all ingress resources across namespaces.

5. ConfigMaps & Secrets

Storage for non-secret and secret configuration
#5
kubectl create configmap app-config --from-file=conf/ Copied!
Create ConfigMap from files.
kubectl create secret generic db-pass --from-literal=password=Pa$$w0rd Copied!
Create a generic secret from literal.
kubectl get secret db-pass -o yaml Copied!
Inspect secret (base64 encoded values).

6. Volumes & Storage

Persistent volumes, PVC and storage classes
#6
kubectl get pv,pvc -A Copied!
List persistent volumes & claims.
kubectl describe pvc my-claim -n web Copied!
Check PVC binding events and storage class.

7. Scaling & Autoscaling

Manual and Horizontal Pod Autoscaler (HPA)
#7
kubectl scale deployment/web --replicas=10 Copied!
Scale a deployment manually.
kubectl autoscale deployment/web --min=2 --max=10 --cpu-percent=70 Copied!
Create HPA based on CPU utilization.
kubectl get hpa -n web Copied!
List HPAs in namespace.

8. Observability & Logs

Logs, exec, top and port-forward for debugging
#8
kubectl logs pod/nginx-123 -n web Copied!
View container logs (for single container pod).
kubectl logs deploy/web -n web --since=1h Copied!
Tail logs for a deployment for the last hour.
kubectl top pods -n web Copied!
Show pod CPU/memory metrics (metrics-server required).
kubectl port-forward svc/web 8080:80 -n web Copied!
Expose service locally for inspection.

9. Debugging & Troubleshooting

Pod troubleshooting, events and exec
#9
kubectl get events -n web --sort-by=.metadata.creationTimestamp Copied!
Show recent cluster events (sorted).
kubectl exec -it pod/nginx-123 -n web -- /bin/sh Copied!
Execute shell inside container for inspection.
kubectl cp /local/file pod/nginx-123:/tmp -n web Copied!
Copy files to/from a pod.

10. Maintenance & Security

Node upgrades, RBAC and image policies
#10
kubectl cordon node-1 Copied!
Mark node unschedulable for maintenance.
kubectl drain node-1 --ignore-daemonsets --delete-local-data Copied!
Evict pods and prepare node for maintenance.
kubectl create rolebinding read-pods --clusterrole=view --serviceaccount=web:deployer -n web Copied!
Quick RBAC binding example.
