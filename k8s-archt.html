1. The Architecture

A detailed look at the Control Plane, Worker Nodes, and essential Addons as defined in the official documentation.

CONTROL PLANE

kube-apiserver

The "Front Door". Exposes the Kubernetes API. The only component that communicates directly with etcd.

etcd

Consistent, highly-available key-value store for all cluster data.

kube-scheduler

Watches new Pods and assigns them to Nodes based on constraints.

kube-controller-manager

Runs core loops (Node, Job, ServiceAccount) to reach desired state.

cloud-controller-manager

Embeds cloud-specific logic. Links cluster to cloud provider APIs (LBs, Routes).

The "Captain". Agent running on every node. Registers node & manages Pods.

kube-proxy

Maintains network rules (iptables/nftables) for Pod communication.

2. The Evolution (v1.27 - v1.34)

A chronological look at major features, code names, and shifts in the ecosystem.

v1.27 "Chill Vibes" (Apr 2023)

  • Registry Freeze: Shifted from k8s.gcr.io to registry.k8s.io.
  • Seccomp Default: Security hardening moved to Beta.
  • In-place Resize (Alpha): First look at resizing pods without restart.

v1.28 "Planternetes" (Aug 2023)

  • Sidecar Containers (Alpha): Native support for auxiliary containers.
  • pkgs.k8s.io: New community-owned package repositories.

v1.29 "Mandala" (Dec 2023)

  • ReadWriteOncePod (GA): Safer storage access mode.
  • Gateway API v1.0: The modern replacement for Ingress hits GA.

v1.30 "Uwubernetes" (Apr 2024)

  • Node Log Query (Beta): View systemd logs via kubectl.
  • Container Resource Autoscaling: HPA scales on container metrics.

v1.31 "Elli" (Aug 2024)

  • AppArmor (GA): Standardized security profiles.
  • Traffic Distribution (Beta): Better routing control for Services.

v1.32 "Penelope" (Dec 2024)

Auto-Remove PVCs

StatefulSets can finally clean up their own PVCs upon deletion. No more orphaned storage costs.

Windows Parity

Graceful Node Shutdown now supported on Windows nodes.

Job 'managedBy'

External controllers (like Kueue) can manage Jobs for AI/Batch workloads.

v1.33 "Octarine" (Apr 2025)

Sidecars Stable

RestartPolicy for init containers is GA. Essential for Service Meshes & Logging agents.

In-Place Resize (Beta)

Resize CPU/RAM without restarting pods. Enabled by default. Game changer for Java/DBs.

v1.34 "Of Wind & Will" (Aug 2025)

DRA Stable

Dynamic Resource Allocation is the new standard for GPUs/FPGAs. Replaces Device Plugins.

Volume Attributes

Modify IOPS/Throughput of volumes on the fly via VolumeAttributesClass.

KYAML

"Kubernetes Dialect" of YAML to fix quoting/ambiguity issues (The Norway Problem).
